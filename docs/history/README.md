# Documentation History Archive

This directory contains implementation history: completed plans, old design notes, migration
records, task backlogs, investigation notes, validation ledgers, and build-system implementation
logs. These files are retained for traceability, not as the first source of truth for current setup
or operation.

Use the current docs before using archive material:

- Operator and repository docs: [`../README.md`](../README.md)
- Build-system references: [`../../build-tools/docs/README.md`](../../build-tools/docs/README.md)
- Contributor workflow: [`../handbook/README.md`](../handbook/README.md)
- Validation workflow: [`../../TESTING.md`](../../TESTING.md)

## Layout

- [`plans/`](plans/): completed or superseded PR plans and gap trackers.
- [`designs/`](designs/): old design notes and design archaeology.
- [`migrations/`](migrations/): rename, prefix, and structure migration notes.
- [`tasks/`](tasks/): historical task backlog snapshot.
- [`build-system/`](build-system/): build-system plans, experiments, roadmaps, and old logs.
- [`investigations/`](investigations/): debugging and performance investigations.
- [`ledgers/`](ledgers/): validation or integration debt ledgers.
- [`process/`](process/): historical process notes.
